\begin{abstract}

	\noindent O presente trabalho tem como objetivo a apresentação de um algoritmo meta-heurístico, utilizando a busca tabu, para o Problema do Caixeiro Viajante (PCV) bem como os resultados computacionais dos testes com as instâncias da literatura.

\noindent \textbf{Palavras-Chave:} PCV, Caixeiro Viajante, Algoritmo meta-heurístico, busca tabu

\end{abstract}
